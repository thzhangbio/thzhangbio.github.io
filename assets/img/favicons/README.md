# Favicon 图标目录

## 如何设置 Favicon

1. **准备图标源文件**
   - 准备一张正方形图片
   - 建议尺寸：512x512 像素或更大
   - 格式：PNG、JPG 或 SVG

2. **使用在线工具生成 Favicon**
   - 访问 [Real Favicon Generator](https://realfavicongenerator.net/)
   - 上传您的图标图片
   - 保持默认选项或根据需要调整
   - 生成并下载图标包

3. **下载并替换文件**
   - 解压下载的文件
   - **重要**：删除 `site.webmanifest` 文件（Chirpy 不使用此文件）
   - 将剩余的图标文件（如 `.PNG`、`.ICO` 和 `.SVG`）复制到此目录中

4. **生成的文件通常包括**：
   - `android-chrome-192x192.png`
   - `android-chrome-512x512.png`
   - `apple-touch-icon.png`
   - `favicon-16x16.png`
   - `favicon-32x32.png`
   - `favicon.ico`
   - `mstile-150x150.png`
   - `safari-pinned-tab.svg`

完成以上步骤后，刷新浏览器即可看到新的 favicon。
