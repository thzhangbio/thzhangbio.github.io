# 图片资源目录

## 头像设置

请将您的个人头像图片放在此目录下，命名为 `avatar.png`（或 `avatar.jpg`、`avatar.svg`）。

**建议规格**：
- 尺寸：512x512 像素或更大
- 格式：PNG、JPG 或 SVG
- 形状：正方形（最佳显示效果）

在 `_config.yml` 中设置头像路径：
```yaml
avatar: /assets/img/avatar.png
```

## Favicon 设置

Favicon 图标文件应放置在 `favicons/` 子目录中。请参考 `favicons/README.md` 了解如何生成和放置 favicon 文件。
